postakod
========

Türkiye'nin tüm posta kodları listesini içeren MySQL veri tabanı

###Toplam
* 81 İl,
* 957 İlçe,
* 52.835 Semt / Mahalle / Köy,
* 3.226 Posta kodundan oluşmaktadır.


Kullanım ve detaylı bilgi için: http://beltslib.net/turkiye-posta-kodlari.html
